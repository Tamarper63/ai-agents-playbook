Use this chooser when you need a strict “facts-only” output contract but must decide **what evidence is allowed**.

## Option 1 — Artifacts-only (no external sources)
Use when you have logs/files/screenshots and you want **zero web knowledge**.

- **Policy:** [Facts-only: Artifacts-only]({{ '/policies/facts-only-artifacts-only/' | relative_url }})
- **Prompt:** [facts-only-artifacts-only.system.txt]({{ '/prompts/facts-only-artifacts-only.system.txt' | relative_url }})
- **Procedure:** [Facts-only: Artifacts-only — procedure]({{ '/how-to/facts-only-artifacts-only/' | relative_url }})

## Option 2 — Authoritative sources required (citations required)
Use when you want externally verified answers with **formal citations** (DOI/standard-id/ISBN/section).

- **Policy:** [Facts-only: Authoritative sources required]({{ '/policies/facts-only-authoritative-sources-required/' | relative_url }})
- **Prompt:** [facts-only-authoritative-sources-required.system.txt]({{ '/prompts/facts-only-authoritative-sources-required.system.txt' | relative_url }})
- **Procedure:** [Facts-only: Authoritative sources required — procedure]({{ '/how-to/facts-only-authoritative-sources-required/' | relative_url }})

## Option 3 — Evidence-Gated Academic Mode (EGAM)
Use when you need the same “authoritative sources required” boundary **plus** an academic register + VERIFIED/NOT VERIFIED sections + confidence scoring.

- **Policy:** [Evidence-Gated Academic Mode (EGAM)]({{ '/policies/evidence-gated-academic-mode/' | relative_url }})
- **Prompt:** [evidence-gated-academic-mode.system.txt]({{ '/prompts/evidence-gated-academic-mode.system.txt' | relative_url }})
- **Procedure:** [Evidence-Gated Academic Mode — procedure]({{ '/how-to/evidence-gated-academic-mode/' | relative_url }})
