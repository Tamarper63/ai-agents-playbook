Stable definitions, naming conventions, and reusable anchors for consistent terminology and linking across **Policies**, **How-to guides**, **Articles**, and **Prompt blocks**.

Use **Reference** for material that should remain stable over time:
- **Definitions / terminology** that multiple pages rely on
- **Naming + linking conventions** (slugs, casing, section names)
- **Reusable reference anchors** that other pages should link to (define once; reuse)

Use **Articles** for long-form arguments/analysis, and **How-to** for step-by-step procedures.
