# Evidence and citation conventions

This page defines internal conventions for citing evidence in this repository.

## Artifact citations (Mode A)

Format:
- `[artifact-id §locator]`

Where:
- `artifact-id` is a filename or a user-provided identifier.
- `locator` is a section / line range / page / figure / snippet label.

## External citations (Mode B)

Provide a stable, human-verifiable locator for each claim, such as:
- DOI for papers
- Standard identifiers + section numbers
- ISBN + edition + section
- Institution + document/page + locator
