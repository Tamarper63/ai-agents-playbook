# Hallucination mitigation — verification methods

This repository uses **“verification methods”** as a local grouping for techniques that add an explicit verification step intended to reduce **confabulation / hallucination** in LLM outputs.

> Note: Academic literature commonly uses broader terms such as **“mitigation methods”** for hallucination reduction.

## Techniques documented here

- [Chain-of-Verification (CoVe)](chain-of-verification/) — a draft → verify → revise procedure (Dhuliawala et al., 2023/2024).
