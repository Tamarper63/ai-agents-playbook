## Usage rules

- **Policy pages are normative**: they define mandatory rules and constraints.
- **Prompt blocks are operational**: copy/paste templates aligned to the policies/guides.
- If a guide and a prompt block diverge, treat the guide as out-of-date and fix it to match the policy.
